Структура чек-листа:
- Name -название чек-листа
- ID - уникальный индификатор
- Build -версия сборки
- Environment - окружение
- Data - дата проведения теста
- Tester - фио тестировщика
- Test type - тип теста
- Checking - названия/список проверок
- Result - результат(статус) проверки
- Comments - коментарии
